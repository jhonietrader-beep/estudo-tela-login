Nome do Projeto

Tela de Login

Descrição

Projeto desenvolvido para praticar conceitos de HTML, CSS e JavaScript. A aplicação simula uma tela de login com validação de campos e interação com o usuário.

Funcionalidades
Validação de usuário.
Validação de senha.
Mensagens de alerta para campos vazios.
Interface responsiva.
Manipulação de eventos com JavaScript.
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

Manipulação do DOM.
Eventos JavaScript.
Validação de formulários.
Estruturas condicionais.
Organização de código.
Boas práticas de desenvolvimento.
